1) Modulo de Plan

vamos criar um projeto de calculo de media dos alunos de uma faculdade. Temos varios cursos, diciplinas, alunos e professores. Cada diciplina tem duas notas e deve ser calculado a media de cada nota. media >= 7 aprovado. Media >= 5 e < 7 exame. Nota < 5 reprovado.

Para desenvolver este projeto vamos utilizar o framework BMad IA. Para isso temos que criar / atualizar os arquivos que estao nas pastas _bmad, _bmad-output e .cursorrules

Vamos usar como tecnologia java 21, maven, banco de dados sqlite, testes automatizados, end-points para entrada e saida de dados, JavaEE Jakarta, JPA.
